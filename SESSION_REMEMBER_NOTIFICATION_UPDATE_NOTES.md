# Session, Remember Me, Forgot Password and Notification Readability Update

This update is intentionally limited to the requested areas:

- Makes sidebar notification messages more readable with high-contrast cards.
- Activates the Remember me checkbox on the login screen.
- Activates Forgot password so Admin receives an in-app notification to review User Management and force a password change/reset.
- Extends the default session timeout from 60 minutes to 720 minutes.
- Adds optional remember-me server session expiry for 30 days.
- Keeps the active signed-in workspace section in the current URL so browser refresh restores the same page.

No procurement workflow, role permissions, users, passwords, purchase requests, gateway pass logic, finance logic, or SQLite records were intentionally changed.
